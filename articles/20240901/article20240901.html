map entryの配列から特定のキーを持つデータのvalueを取得する方法

Created at : 2024/09/01

環境

DartPadで実行

  • Dart SDK 3.5.1
  • Flutter SDK 3.24.1

サンプルコード

特定のキーを持つデータのvalueを取得する 
List> mapList = [
  {'id': 1, 'name': 'Alice'},
  {'id': 2, 'name': 'Bob'},
  {'id': 3, 'name': 'Charlie'}
];

// idが2のnameを取得する
var value = mapList.firstWhere((map) => map['id'] == 2)['name'];
print(value); // Bobが表示される

条件に合致する要素が存在しない場合には、firstWhereメソッドにはオプションのorElse関数を提供して、デフォルトの値を設定することもできる

orElseでnullを設定する 
var value = mapList.firstWhere(
  (map) => map['id'] == 4,
  orElse: () => {'name': null}
)['name'];
