SwiftUI - Text : 表示しきれなかったときの設定

Created at : 2024/10/20

環境

  • Xcode 15.4

主なモディファイア

  • .lineLimit
    • 表示する行数の上限を指定
  • .truncationMode
    • テキストが表示しきれない場合、どの位置で省略するかを指定
      • .head: 先頭を省略する
      • .middle: 真ん中を省略する
      • .tail: 末尾を省略する
  • .allowsTightening
    • テキストがレイアウトに収まらないときに、テキストの文字間を狭めて表示するかどうかを指定

サンプルコード

行数制限をかけて末尾に省略記号を表示 
Text(longText)
  .lineLimit(1) // 1行に制限
  .truncationMode(.tail) // 末尾を省略
複数行で表示 
Text(song.title)
  .lineLimit(2) // 最大2行まで表示
  .truncationMode(.tail) // 2行目の末尾を省略
先頭を省略 
Text(longText)
  .lineLimit(1) // 1行に制限
  .truncationMode(.head) // 先頭を省略
末尾を省略してタイトに表示 
Text(song.title)
  .lineLimit(1)
  .truncationMode(.tail) // 末尾を省略
  .allowsTightening(true) // 必要に応じて文字間を詰める
