MacBookでFFmpegを使用する

Created at : 2021/08/12

waveファイルをmp3に変換したくて、FFmpegをMacbookにインストールしました。 そのときのメモです。

前提

Homebrew がインストールされていること

手順

1. FFmpegをインストールする

ターミナルを開き、次のコマンドを入力してEnterキーを押下

brew install ffmpeg

2. WAV → MP3 変換

ターミナルを開き、次のコマンドを入力してEnterキーを押下

ffmpeg -i input.wav output.mp3

その他

FFmpegをアップデートする

ターミナルを開き、次のコマンドを入力してEnterキーを押下

brew update ffmpeg

FFmpegをアンインストールする

ターミナルを開き、次のコマンドを入力してEnterキーを押下

brew uninstall ffmpeg
